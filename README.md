# Activity Tracker

## 📌 Objective
To implement front-end interactivity that allows users to update activity status and view real-time progress using JavaScript.

## 🎯 Problem Focus
This project converts a static UI into an interactive application by enabling users to mark activities as completed and dynamically update progress without reloading the page.

## 🖥️ Application Overview
This is a simple Activity Tracker web application where users can:
- View a list of activities
- Mark activities as completed
- Track progress in real-time

## 📊 Data Handling Approach
Activities are stored using a JavaScript array. No backend is used. All updates happen dynamically in the browser.

## 🎨 UI Components
- Activity List
- Action Buttons (Mark as Done)
- Progress Display

## ⚙️ Interactivity Implementation
- Event handling using JavaScript
- Toggle activity status (Pending → Completed)
- Instant UI updates without page reload

## 🧠 Application Logic
- Counts completed activities
- Updates progress text dynamically
- Reflects changes immediately

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ✨ Key Features
- Mark activities as completed
- Real-time progress tracking
- Dynamic UI updates

## 📦 Output
A working front-end application that tracks activities and updates progress instantly.

## 📚 Learning Outcome
- JavaScript event handling
- DOM manipulation
- Dynamic UI updates
- Basic state management

## 🔚 Conclusion
This project successfully demonstrates how to build an interactive front-end application using basic JavaScript concepts.
